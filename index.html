Tentang Sistem

Platform monitoring ini dibangun untuk membantu Anda mengawasi perangkat IoT dengan mudah. Dengan integrasi Firebase, semua data tersinkronisasi secara realtime tanpa perlu konfigurasi rumit. Tersedia mode demo mandiri dan dukpenuh untuk ESP8266/ESP32.

  • ✔ Mudah diintegrasikan dengan ESP32 / ESP8266
  • ✔ Mendukung banyak sensor secara bersamaan
  • ✔ Dashboard yang dapat dikustomisasi
